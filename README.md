# Manual de Conformidade e Comunicação Segura

## Grupo Positivo Brasil - Instagram e Facebook

Este manual foi desenvolvido para orientar a equipe de marketing do Grupo Positivo Brasil sobre como comunicar serviços de limpeza de CPF, aumento de score de crédito e blindagem de dados no Instagram e Facebook de forma segura, legal e eficaz.

### Conteúdo

- Introdução e Contexto Regulatório
- Políticas da Meta para Serviços Financeiros
- Glossário de Gatilhos (palavras, hashtags, imagens proibidas)
- Legislação Brasileira Aplicável (CONAR, Lei de Defesa do Consumidor, BACEN)
- Casos Reais de Bloqueio com Lições Aprendidas
- Diretrizes Visuais e de Conteúdo
- Estratégias de Marketing Saudável
- Checklist de Conformidade Pré-Publicação
- Quiz de Avaliação

### Como Usar

Abra o arquivo `index.html` em seu navegador. O manual é responsivo e funciona em desktop, tablet e mobile.

### Versões

- `index.html` - Versão Web (desktop)
- `index_mobile.html` - Versão Mobile (otimizada para celulares)
